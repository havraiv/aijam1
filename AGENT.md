# Agent Entry Point

Primary workspace instructions are maintained in `.github/copilot-instructions.md`.

If you are an AI coding agent working in this repository:

- Load and follow `.github/copilot-instructions.md` first.
- Use `npm install`, `npm run dev`, `npm run build`, and `npm run lint` for validation.
- Treat `src/App.tsx` as the composition root for page sections.
- Keep shared section data in `src/content.ts`.
